Movie web app

Opening screen

On the opening screen we have the option to register or connect to the application, including sending an email to the user and uploading a profile picture. All the data are saved in the DB Browser (SQLite).

The main screen

A main screen with lots of options like adding a movie, get a review of the movie from the app, add a comment to the movie as a user and play a movie trivia game.

Technologies and link to GitHub

Link to the project in github.
  • Connection to the API of IMDB and GPT chat.
  • Data development in DB Browser (SQL lite).
  • Flask (web framework).
