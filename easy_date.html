Easy Date app

The main screen

On the opening screen you can see the profiles that are in the application. You can click on like or reject. If the profiles click on like for each other, a match will be created. In addition, we will have a settings screen for choosing a profile according to the distance in km from the city where he lives and according to age. In addition, you can edit our profile details and correspond in live Chat with our match.

Technologies and link to GitHub

Link to the project in github.
  • Connection to the API of Google Maps and GPT chat.
  • Data development in DB Browser (SQL lite).
  • Flask (web framework).
  • Design in Html and CSS.
  • Using phone verification with TWILIO.
  • Creating a live chat using python socketio documentation.
